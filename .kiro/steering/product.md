# Product Overview

## CardVault - Secure Offline Digital Wallet

CardVault is an **offline-first Android application** that serves as a digital repository for physical cards (credit, debit, gift cards, vouchers, memberships, etc.). The app prioritizes complete data locality, premium UI/UX with smooth animations.

### Core Principles
- **100% Offline Operation**: No network permissions, no cloud dependencies, all data stays on device
- **Security First**: AES-256-GCM encryption with Android Keystore, biometric authentication
- **Premium Experience**: 60fps animations, 3D card flip effects, Material Design 3
- **Smart Scanning**: Camera-based card capture with ML Kit OCR for automatic data extraction
- **Flexible Organization**: Custom categories, card types, and fields

### Key Features
- Offline-only card storage and management
- Camera scanning with automatic text recognition
- 3D card flip animations and smooth transitions
- Custom card categories with icons and colors
- No analytics, tracking, or data collection

### Target Users
- **Frequent Travelers**: Need quick access to travel cards without connectivity
- **Deal Hunters**: Organize loyalty cards, vouchers, and track expiry dates
- **Privacy-Conscious Users**: Want complete control over their card data

The app follows Android architecture best practices with MVVM pattern, Jetpack Compose UI, and a layered reactive architecture.
